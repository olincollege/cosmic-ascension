Cosmic Ascension Video Game

Project Summary

The goal of this project is to create an interactive program using the Python coding language and the Model-View-Controller design pattern. We created a single player, platformer video game. The player controls a spaceship, with the objective to make it as far from Earth (the bottom) as possible. If the character falls off a platform, they have ended their mission and have to start again. There are 3 difficulties for players to choose from, easy, medium, and hard. These are dictated by distance between the platforms.

Running Program and Gameplay

To run our program and play the game, first clone the repository at the following link.

Cosmic Ascension Repo

Additionally, install Pygame by running pip install pygame in your terminal.

Finally, play the game by running the main.py file.

Game Controls

The character sprite is controlled with 3 keyboard keys.

Left arrow for moving left

Right arrow for moving right

Space bar to jump
